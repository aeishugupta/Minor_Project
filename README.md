# 📊 Group DNA Analysis Project

## 📌 Project Overview

The **Group DNA Analysis Project** is a Python-based data analysis project that analyzes exported group chat data and extracts meaningful insights about participant behavior and communication patterns. The project uses Python programming concepts such as loops, dictionaries, string processing, conditional statements, and NumPy arrays to perform chat analytics.

This project aims to identify user activity patterns, message statistics, personality traits, and group behavior trends from chat data.

---

## 🎯 Objectives

* Analyze group chat activity using Python.
* Identify the most and least active participants.
* Calculate media and deleted message statistics.
* Detect night owl behavior patterns.
* Find the most frequently used words.
* Analyze daily and hourly activity trends.
* Create a participant activity matrix using NumPy.
* Generate personality tags based on chat behavior.

## 📂 Features Implemented

### 1. Total Message Analysis

* Counts total messages sent by each participant.

### 2. Most Active User

* Identifies the participant with the highest number of messages.

### 3. Least Active User

* Identifies the participant with the lowest number of messages.

### 4. Media Message Analysis

* Counts the total number of media messages shared in the group.

### 5. Deleted Message Analysis

* Counts the total deleted messages.

### 6. Night Owl Detection

* Identifies the participant who sends the most messages between late-night hours.

### 7. Word Frequency Analysis

* Calculates the most frequently used word in the chat.
* Displays the top 10 most used words.

### 8. Daily Activity Analysis

* Finds the busiest day and least active day in the chat.

### 9. NumPy Activity Matrix

* Creates a 6 × 24 activity matrix.
* Rows represent participants.
* Columns represent hours of the day.

### 10. Personality Analysis

The project assigns personality tags based on chat behavior:

* 🦉 Night Owl
* 📢 Spammer
* 👻 Ghost

### 11. Final Group DNA Report

* Generates a summarized report containing all major analytics results.
